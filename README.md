# Exicore — Construction Robotics Platform

Full-stack, vendor-agnostic platform helping construction companies discover, evaluate, simulate, procure and control construction robots.

## Phase 1 (Current)
- Robot Discovery Catalog — 60+ vendors, searchable and filterable
- Robot Detail Pages — full vendor profiles, API status, funding
- BIM Simulation Interface — visualize robots on your project

## Tech Stack
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js / FastAPI
- Database: Supabase
- Deployment: Vercel

## Development
```bash
npm install
npm run dev
```
